# AWMC BOT进阶教程

## xx完成表/xx进度

查询将/极/神/舞舞牌子获取进度的指令。
其中，`xx完成表`指令可以输出图片，而`xx进度`指令可以输出文字信息。

## 发票 [x]

::: warning
发票有风险，操作需谨慎

BOT对发票可能造成的负面影响概不负责。

参见[AWMC BOT 服务协议与隐私政策](/guide/bot/terms#十、免责声明)第十条。
:::

向游戏账号发放功能票，其中`[x]`为你所期望的功能票倍数，取值范围为2-6的整数，未填写则默认为2。

4倍及以上功能票发放易失败。

## 我要上[x]分

在B35与B15中，分别推荐5首可用来推分的歌，其中`[x]`为你期望推的分数，也可留空。
