# AWMC BOT 进阶教程

::: tip 提示
善用 <kbd>Ctrl+F</kbd> 进行查询。
:::

## 一、基础与帮助

| 命令 | 说明 |
|------|------|
| `帮助maimaiDX` / `帮助maimaidx` | 发送帮助图 |
| `项目地址maimaiDX` | 项目 GitHub 链接 |
| `更新maimai数据` | **超管**：更新曲库/谱面/别名等本地数据 |
| `今日mai` / `今日舞萌` / `今日运势` | 今日人品 + 推荐曲 |
| `mai什么` / `mai什么歌` | 随机推荐一曲（可带推分意图） |
| `给个绿14+` 等 | 按难度/类型随机抽曲 |
| `查看排名` / `查看排行` | 全服 Rating 排行榜 |
| `我的排名` | 自己的 Rating 排名 |
| `主题` / `theme` | 切换 B50 等图片主题 |

## 二、查歌 / 搜歌

| 命令 | 说明 |
|------|------|
| `查歌 <关键词>` | 按曲名搜索 |
| `定数查歌` | 按定数范围查歌 |
| `bpm查歌` | 按 BPM 查歌 |
| `曲师查歌` | 按曲师查歌 |
| `谱师查歌` | 按谱师查歌 |
| `<别名>是什么歌` | 别名反查，生成详细信息图（含谱面标签、定数变化、预览链接等） |
| `id12345` | 查指定 ID 曲目详情（含谱面标签、定数变化、预览链接等） |
| `谱面12345紫` | 谱面预览链接 |
| `提取曲绘 <ID>` | 提取曲绘图片 |

## 三、成绩图（B50 系列）

### 常规

| 命令 | 说明 |
|------|------|
| `b50` | 标准 B50（B35+B15 分组） |
| `ab50` / `a50` | 无视分组取前 50 |
| `合作b50` / `合作a50` | @好友 双人合作 B50 |
| `lxb50` / `落雪b50` | 强制落雪数据源 B50 |

### 筛选 / 变体

| 命令 | 说明 |
|------|------|
| `紫b50` / `13+b50` / `Master b50` 等 | 按难度筛选 B50 |
| `紫ab50` 等 | 按难度筛选 ab50 |
| `镜代b50` 等 | 按版本代筛选 B50 |
| `l镜代b50` / `l爽代b35` 等 | 历代版本定数重算 B50/B35 |
| `dx2026b35` | 彩代 B35 |
| `fcb50` / `fcallb50` | FC 向 B50 |
| `apb50` / `apallb50` | AP 向 B50 |
| `拟合b50` / `拟合b50全部` | 按拟合定数排序 |
| `寸b50` / `寸ab50` | 寸止（踩评级线）成绩 |
| `锁血b50` / `锁血ab50` | 锁血（踩评级下限）成绩 |
| `越级b50` / `越级ab50` | 越级成绩 |
| `理想b50` / `理想ab50` | 评级升一档后的理想 B50 |
| `含金量` / `含水量` | 成绩含金量/含水量分析图 |

### 分析 / 对比

| 命令 | 说明 |
|------|------|
| `我有多菜` | Rating 分布柱状图 |
| `底力分析` | B50 标签雷达图 + 条形图，仅供参考 |
| `minfo` / `info` | 单曲各难度游玩详情 |
| `ginfo` | 单曲全服统计 + 饼图 |
| `分数线` | 计算指定达成率容错 TAP/GREAT 数 |

## 四、定数表 / 完成表 / 进度

| 命令 | 说明 |
|------|------|
| `14定数表` | 查看定数表（lv7–15） |

## 六、落雪（Lxns）数据源

| 命令 | 说明 |
|------|------|
| `lxbind` / `绑定落雪` | OAuth 绑定落雪查分器 |
| `lxunbind` / `解绑落雪` | 解绑 |
| `数据源 水鱼` / `数据源 落雪` | 切换 B50 等默认数据源 |

::: info 说明
B50 系列按用户偏好自动走水鱼/落雪；部分功能（拟合、含金量等）落雪不支持时会回退水鱼。
:::

## 七、PC 数（机台）

PC = Play Count，即每首歌在机台上的游玩次数。

| 命令 | 说明 |
|------|------|
| `更新pc数 <二维码>` | 机台登录并同步 PC 数 |
| `我的pc数` | 个人 PC 数 Top 列表 |
| `pc排行` | PC 数排行榜 |
| `pc数 <歌曲ID>` | 指定曲目 PC 数 |
| `pc50` / `pca50` | 按 PC 数排序的 B50 图 |

## 十、谱面印象（PMYX）

| 命令 | 说明 |
|------|------|
| `谱面印象 <ID>` | 查看谱面印象 |
| `写谱面印象` | 发表印象 |
| `回复谱面印象` | 回复他人印象 |
| `点赞谱面印象` | 点赞 |

::: tip 提示
使用 `查歌`（ID 或别名）查询时，会**自动附带**谱面印象、标签、定数变化、预览链接等合并转发。
:::

## 附：关于「拍一拍」

BOT 对 QQ 的拍一拍会做出反馈。若 BOT 是管理，可能触发随机时长的**禁言**（最高 1 天），拍之前请三思。

## 附：发票

::: danger 警告
发票有风险，操作需谨慎。AWMC TEAM 对发票可能造成的负面影响概不负责。

参见 [AWMC BOT 服务协议与隐私政策](/guide/bot/terms#十、免责声明) 第十条。
:::

使用 `发票 [x]` 向游戏账号发放功能票，`[x]` 为功能票倍数（取值范围 2-6 的整数，默认 2）。4 倍及以上功能票发放易失败。
