# 术语解释

::: tip 提示
善用Ctrl+F进行查询
:::

## B50

全称Best 50。

舞萌DX通过一个特殊的公式来计算每首歌获得的Rating。

从大到小排列，其中前35个<abbr title="非当前版本更新的歌曲">旧版本歌曲</abbr>和前15个<abbr title="国服现行版本为舞萌DX 2025">新版本歌曲</abbr>组成了B50。

将它们的Rating相加，便得出<abbr title="国服在舞萌DX 2025后翻译为DX评分">DX Rating</abbr>。

### B40

全称Best 40。

在舞萌DX 2022及之前的版本采用的<abbr title="国服在舞萌DX 2025后翻译为DX评分">DX Rating</abbr>计算方法，现已不再沿用，但部分BOT仍可生成。

## FC/全连

全称<abbr title="国服在舞萌DX 2025后翻译为全连击">Full Combo</abbr>，是舞萌特殊评价的一种，需要谱面游玩过程中所有的Note都取得Good及以上评价。

### FC+

全称<abbr title="国服在舞萌DX 2025后翻译为全连击+">Full Combo+</abbr>，是舞萌特殊评价的一种，与FC不同的是，需要谱面游玩过程中所有的Note都取得Great及以上评价。

## AP/PM/AJ

::: tips 提示
这种评价在音游中广泛存在。
:::

全称<abbr title="国服在舞萌DX 2025后翻译为完美无缺">All Perfect</abbr>，是舞萌特殊评价的一种，需要谱面游玩过程中所有的Note都取得Perfect及以上评价。

PM的说法来自移动端音游 Arcaea ，全称Pure Memory，与AP的条件类似，AJ的说法则来自街机音游 中二节奏 ，全称All Justice，与AP的条件类似。

### AP+/理论

全称<abbr title="国服在舞萌DX 2025后翻译为完美无缺+">All Perfect+</abbr>，是舞萌特殊评价的一种，需要谱面游玩过程中所有的Note都取得Perfect及以上评价，且所有BREAK音符均取得Critical Perfect评价，达成后的分数为<abbr title="101.0000%">理论最高值</abbr>，因此也被称为理论。
